Directories:
 - App: Core of the platform
 - Helper: Small classes to manage simple tasks
 - Libs: Complex PHP Libraries
 - Model: Database Model
 - resources: Internal resources
 - Services: Interfaces for use PHP Libraries
 - Systems: Systems of the application
    - <SystemName>
        - _base: Directory for basic stuff like Firewall, Preloader, base Template
        - _config: Configuration and router for the system
        - <ModuleName> Directory for each module
            - controller: For controllers
                - helper: For services
            - view: For html templates
                - i18n: For internationalization
                - email: For emails templates
                - pdf: For pdf templates
