CSRFHandler
in package
  • Arkit

CSRFHandler.php : 13

This class handle the CSRF token.

  • Generate and validate a hidden input code given a Form ID and expiration time
  • Generate and validate a cookie given a Form ID. The cookie name is associated with the Form ID. So, the unique who know the expected cookie is the server The cookie value is associated with the Session and the Form ID.
